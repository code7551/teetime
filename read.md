This project is Golf Academy Management name 'Teetime golf center (TTGC)'


This project have 3 group of user
1.Owner: Web App
2.Pro: Web App
3.Student: Line Official Account

Requirement of the project
- Owner able to create 'Pro' and 'Student'
  - Review payment transaction and automatic add student hourly
  - See All timetable of each students
  - View all learning history
  - Pro profile page
    - show students
    - show learning history
  - Student profile page show like 'Pro'
  - Payment Dashboard

- Pro able to manage 'Student' and receive commission 40% each class
  - See timetable of each student under manage
  - Submit review and short video after student learn
  - View income and payment history each transaction able to see fully info like 'Teach student xxx' received 'xxx' bath
- Student able to 
  - view remaning hour
  - view their leaning history
  - view Pro commented on that day or short video (if exists)
  - receive line rich message notification after Pro commented
  - Submit pamynet receipt for course renewal

Stack prefered
- NextJS + Typescript + Tailwind
- Reacthook form, heroui
Database
- Firestore database
Storage
- Cloudflare R2


please create .env.example for each credential
